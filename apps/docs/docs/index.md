# Basilisk Documentation

**Basilisk** is a Cardano analytics platform — real-time on-chain data, portfolio tracking, and the first **x402 agent-native** trading layer on Cardano. These docs cover everything you need to use the app, integrate with the REST API, or wire an AI agent into the network through MCP and x402 micropayments.
