# @basilisk/api-gateway

API gateway / BFF — auth, rate-limiting, caching, API keys, and the public Data API.

**Owns:** EPIC-1 (US-1.3), EPIC-8 (US-8.1/8.2/8.3)

This is a scaffold. Implementation tracked via the `US-*` GitHub issues for the epics above. See [`docs/BASILISK_MVP_PLAN.md`](../../docs/BASILISK_MVP_PLAN.md).

```bash
npm run dev --workspace @basilisk/api-gateway        # watch mode
npm run typecheck --workspace @basilisk/api-gateway
```
