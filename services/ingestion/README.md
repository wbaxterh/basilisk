# @basilisk/ingestion

Chain follower — tails Ogmios/Oura, normalizes events, publishes to Redis, persists swaps.

**Owns:** EPIC-0 (US-0.1, US-0.2)

This is a scaffold. Implementation tracked via the `US-*` GitHub issues for the epics above. See [`docs/BASILISK_MVP_PLAN.md`](../../docs/BASILISK_MVP_PLAN.md).

```bash
npm run dev --workspace @basilisk/ingestion        # watch mode
npm run typecheck --workspace @basilisk/ingestion
```
