# @basilisk/alerts

Alert rules engine evaluating the live event stream (price, %move, whale, balance, new LP).

**Owns:** EPIC-7 (US-7.1, US-7.4)

This is a scaffold. Implementation tracked via the `US-*` GitHub issues for the epics above. See [`docs/BASILISK_MVP_PLAN.md`](../../docs/BASILISK_MVP_PLAN.md).

```bash
npm run dev --workspace @basilisk/alerts        # watch mode
npm run typecheck --workspace @basilisk/alerts
```
