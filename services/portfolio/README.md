# @basilisk/portfolio

Resolves UTxOs to holdings, marks to market, computes P&L; powers portfolio + wallet profiler.

**Owns:** EPIC-2, EPIC-5, EPIC-6

This is a scaffold. Implementation tracked via the `US-*` GitHub issues for the epics above. See [`docs/BASILISK_MVP_PLAN.md`](../../docs/BASILISK_MVP_PLAN.md).

```bash
npm run dev --workspace @basilisk/portfolio        # watch mode
npm run typecheck --workspace @basilisk/portfolio
```
